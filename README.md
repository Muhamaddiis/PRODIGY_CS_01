## Caesar Cipher GUI Application
<img src="./Assets/img.png">
### This is a graphical user interface (GUI) application built using Python's Tkinter library that allows users to easily encrypt and decrypt messages using the Caesar Cipher algorithm.


## Features:

1. Encrypt and Decrypt Messages: The application supports both encryption and decryption modes.

2. Customizable Shift Value: Users can provide a custom shift value (an integer) to modify the encryption or decryption process.

3. Result Display: The resulting encrypted or decrypted message is displayed in a text box, and can be copied to the clipboard.

4. User-Friendly Interface: The application has a simple, clean GUI built with Tkinter that enables easy interaction.

## Requirements:

Python (version 3.x or later)

Tkinter library (usually comes pre-installed with Python)
